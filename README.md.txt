# AI Prompt Organizer

AI Prompt Organizer is a simple Python project that helps users store, organize, and search prompts by category.

## Features

- Add prompts under categories
- View saved prompts
- Search prompts by keyword
- Store prompts in a JSON file

## Categories Examples

- Marketing
- Church
- Business
- SaaS
- Social Media
- Content Creation

## Technologies Used

- Python
- JSON for data storage

## How to Run

1. Make sure Python is installed
2. Open the project folder in terminal
3. Run:

```bash
python prompt_organizer.py
